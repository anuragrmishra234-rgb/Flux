Flux is a high-performance, context-aware productivity ecosystem designed to streamline the way students and developers research and work.

The project transforms your browser from a messy set of tabs into a focused "Command Center" by synchronizing a web dashboard with a specialized Chrome Extension.

Core Features:
Workspace Management (Contexts):
Create and save "Contexts" (groups of URLs and notes) for specific tasks like "React Development," "Lab Research," or "Exam Prep."
One-Click Launch: Instantly open all required resources in your browser with a single click from the Flux Dashboard.
Focus Mode & Tab Purge:
When activating a workspace, Flux can automatically close unrelated tabs, forcing you into a "Deep Work" state for your current task.
Flux Research (AI Agent):
A built-in AI sub-agent that searches for any subject (e.g., "Docker setup," "Python arrays") and automatically finds the top 3 documentation and tutorial resources for you.
Real-Time Context Heartbeat:
The Chrome extension reports your active tab back to the dashboard, allowing the AI to understand your current focus and provide relevant help in real-time.
Secure Authentication:
A full user system using JWT and MongoDB Atlas, ensuring your workspaces and research history are private and accessible from any PC.
Technical Stack:
Frontend: React (Vite), Tailwind CSS, Framer Motion (for sleek animations).
Backend: Node.js, Express, Socket.io (for real-time sync).
Database: MongoDB Atlas.
Browser: Chrome Extension Manifest v3.
